README!

          This app takes in a sctring via the text panel and replaces certain ascii symbols logical ones. The conversion key is...
          

          "{A}" : ∀
          "{E}" : ∃
          "->"  : →
          "<_>" : ↔
          "not" : ¬
          "not " : ¬
          " and " : ∧
          " or " : ∨
          " xor " : ⊕
          ":=" : ≡
          "{U}" : ∪
          "{u}" : ∩
          "{u>}" : ⊂
          "{ue}": ∈
          "{u=}" :"⊆
          

          If you want to use the words "and", "or", without "not" being converted then put a \ infront of it. For instance "not p or not q := not (p and q) : DeM\organ's Law" will translate into "¬p ∨ ¬q ≡ ¬(p ∧ q) : DeM\organ's Law". Press the Remove "\"s button to remove the special backslash sequences.
